# 📉 PUT OPTION SURVIVAL (풋옵션 트레이딩 서바이벌)

트럼프의 방한/국제 정세 돌발 발표를 앞두고, 15초 안에 시장의 방향을 예측해 살아남는 **파생상품 모의 투자 게임**입니다. 
단기간에 엄청난 수익을 올릴 수 있는 '옵션 매수'의 쾌감과, 한순간의 판단 착오로 전 재산을 잃는 '옵션 매도'의 **무한대 리스크(마진콜)**를 직접 체험해 보세요.

👉 **[게임 플레이하기 (Live Demo)](https://shiene1010.github.io/MyAntigravityTest/)**

---

## 🎮 게임 방법 (How to Play)

1. **상황 파악**:
   - 현재 국제 유가(WTI) 기초자산은 **$100.00**입니다.
   - 시장을 뒤흔들 트럼프 전 대통령의 중대 발표까지 **단 15초** 남았습니다.
   - 당신의 초기 투자 자본금은 **$50,000**입니다.

2. **포지션 배팅 (15초의 승부)**: 
   시간이 다 지나가기 전에 하단의 두 가지 포지션 중 하나를 선택해야 합니다.
   - 🟢 **풋옵션 10계약 매수 (Buy Put)**: 유가가 하락하면 하락한 만큼 무제한 수익을 얻습니다. 만약 유가가 떨어지지 않으면 처음에 낸 프리미엄(-$20,000)만 잃고 끝납니다. (손실 제한, 수익 무대)
   - 🔴 **풋옵션 10계약 매도 (Sell Put)**: 권리를 남에게 팔아 당장 프리미엄(+$20,000)을 현금으로 챙깁니다. **단, 유가가 폭락하면 피할 수 없는 무한대의 빚(마진콜)을 지게 됩니다.** (승률은 높으나 초고위험)
   - *아무것도 누르지 않을 경우*: 타이머 종료 시 관망세(무포지션)로 종료됩니다.

3. **운명의 시간 (The Event)**:
   - 타이머가 **5초** 남았을 때 특정 속보(찌라시)가 화면 하단에 터지며 유가가 약 **15%($100 → $85) 폭락**합니다.
   - 이 폭락 시점에 옵션 매수자와 매도자의 실시간 평가손익(P&L)이 어떻게 요동치는지 잔고 패널을 확인하세요.

4. **결과 확인 (피의 영수증)**:
   - 15초 타이머가 완전히 종료되거나, 옵션 매도자가 유지증거금을 맞추지 못해 **마진콜(강제 청산 SMS 알림)**을 당하면 게임이 끝납니다.
   - 플레이 내용에 따라 세금(11%)이 정산된 최종 **[투자 영수증]**과 AI의 팩트폭력 코멘트를 확인해 보세요!
